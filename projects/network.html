I created a network topology generator using C, Python, BASH, Traceroute and Graphviz. The execution is fully automated using the aformentioned set of technologies.

Let's take the technologies and how they are used one by one.

  • C is used to create a DNS lookup. It returns a IPv4 and IPv6 address given a DNS name - e.g. example.com.
  • Traceroute, a Linux command, is used to track the path of a packet send from the source to a destination IPv4 and IPv6 address.
  • Python is used to format the output from Traceroute into an acceptable format for Graphviz.
  • Graphviz is used to generate an undirected map - i.e. the topology - using the formatted output of Traceroute.
  • Finally, BASH is used to glue all the steps together and fully automated the process. That is, the BASH file can be executed given one or more command line arguments as DNS names, and when it is done, the topology map is generated.

Background

This project was part of a University assignment. As a true programmer, I worked for several hours to automate the process of generating a topology map instead of doing it by hand in a few minutes.
