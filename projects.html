Compiler

I'm following Douglas Thain's book to create my own compiler. This is a project I've wanted to do for some time now but only recently found the courage and motivation to do so, after finding out Thain's book. I am not expecting to built anything too useful but, I do expect it to be a fun learning experience. The final goal would be to build a compiler that can convert code written in my own, custom language, to executable code.
